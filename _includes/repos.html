Argus Recon

Argus automatically enumerates a variety of services.

Functionality

There are modules for:

  • Port Scanning
  • Web App Enumeration
  • SMB Enumeration
  • Kerberos Enumeration
  • DNS Enumeration

Ares Attack

Ares executes simple attacks, or sets up attacks for you.

Functionality

There are modules for:

  • Buffer Overflow (Generate binary payload using specified address & offset)
  • XSS (Generates payload, runs server for cookie exfil, & redirect users)
  • Active Directory attacks (ASREP Roasting & Kerberoasting)
  • Wordlist Mutation (Optionally convert entries to alternate format, like B64)
  • File Uploads (Bypass common defenses)
  • Bruteforcing MSSQL in an AD environment

Artemis Hunter

Artemis generates payloads in various languages, & outputs commands for lateral movement in an Active Directory environment.

Functionality

Reverse Shells

Artemis creates payloads for Windows and Linux machines, and supports the following languages:

  • PowerShell
  • Bash
  • Netcat
  • Java
  • Python
  • PHP

Lateral Movement

Artemis supports the following techniques:

  • WMI
  • WinRM (WinRS and PS Remoting)
  • DCOM
  • PsExec

Hades PrivEsc

Hades performs Post Exploitation recon for standalone or domain-joined Windows machines. Hades & Charon are complementary PowerShell scripts which are designed to function over non-interactive protocols (e.g., WinRM).

Functionality

Hades

The main Hades script identifies PrivEsc vectors & sensitive files. It also collects system information, such as: routing information, Hotfixes, additional drives, etc.

Charon

Charon is a supplemental program for domain-joined machines. It unravels nested groups using LDAP queries, identifies all other computers in the domain, etc.
