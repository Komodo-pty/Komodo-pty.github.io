Ruby Dependencies

[HW 67 & INE C 137]

Like with Python, it's recommended to NOT use sudo or su for installation. You can install gems as the current user like-

gem install --user-install httpclient

Environemnt / PATH

Ensure that gems are installed in a place that the exploit can locate. Make sure that the program can find the dependencies.

gem environment

RVM Sandbox (like Python Venv)

Even tools like bundler will install gems system wide. RVM may be a solution, but can have some downsides.

Troubleshoot

Load Error

The program is having trouble locating the dependecy gem files.
