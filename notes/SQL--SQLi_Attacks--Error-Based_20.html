Error-Based


Background

[OSCP 64]

Sometimes data can be embedded inside the returned error message. This allows you to actually exfiltrate data.

This is distinct from a Blind Error-based vulnerability wherein you can only use the presence of a generic error to determine whether or not a query returns an error.
