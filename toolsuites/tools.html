Argus Recon

Argus is designed to automate basic enumeration for a variety of services.

Functionality

There are modules for:

  • Port Scanning
  • Web App Enumeration
  • SMB Enumeration
  • Kerberos Enumeration
  • DNS Enumeration

Artemis Hunter

Artemis will generate reverse shell payloads in various languages, and provides commands to stabilize those shells. Artemis also can generate commands for lateral movement in an Active Directory environment.

Functionality

Reverse Shells

Artemis creates payloads for Windows and Linux machines, and supports the following languages:

  • PowerShell
  • Bash
  • Netcat
  • Java
  • Python
  • PHP

Lateral Movement

Artemis supports the following techniques:

  • WMI
  • WinRM (WinRS and PS Remoting)
  • DCOM
  • PsExec

Hades PrivEsc

Hades performs Post Exploitation recon for Windows machines. It can be run on a standalone machine, or on a domain-joined machine to scan the Active Directory environment.

Functionality

Hades and Charon are complimentary PowerShell scripts that are designed to work even if your shell is using a non-interactive protocol (e.g. WinRM).

Hades

The main Hades script enumerates searches a Windows machine for a variety of PrivEsc vectors and sensitive files. It also collects system information, including: routing information, installed Hotfixes, additional drives, etc.

Charon

Charon is a supplemental program for domain-joined machines. It unravels nested groups using LDAP queries, identifies all other computers in the domain, etc.

Ares Attack

Ares can be used to execute simple attacks, or to setup attacks for you.

Functionality

There are modules for:

  • XSS (Stealing Cookies)
  • Active Directory Roasting
  • Mutating a wordlist
  • Buffer Overflow (Generating a binary payload file)
  • Bruteforcing MSSQL in an AD environment
