Anaconda Environment (base)

Anaconda is a package manager, Env manager, & Python (+ related packages) distribution. Cross-platform, FOSS Data Science API platform for Python & R programming languages.

(base) Terminal Prefix

Similar to Python's Venv, which prepends (env) to your shell.

The (base) prefix indicates that the default Anaconda Env is active. You can manually deactivate it with:

conda deactivate
