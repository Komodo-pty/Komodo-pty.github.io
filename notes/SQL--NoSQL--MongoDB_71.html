Anonymous Connection

Try running the command without any args to see if it allows Anonymous Connections without any credentials specified.

mongo

Enumeration

Once connected using the mongo client, enumerate dbs (Databases), the collections within each dbs (They're like Tables), and the contents of each Collection.

dbs

Even if they appear empty, a dbs could still contain data!

show dbs

admin 0.000 GB

backup 0.000 GB

config 0.000 GB

Collections

Like with SQL, you first need to connect to a Database.

use backup

Dump Collection's Data

db.user.find()
