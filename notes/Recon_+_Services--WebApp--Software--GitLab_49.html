GitLab


Background

Can run local instance of Gitlab.

def creds allegedly admin@local.host:5iveL!fe

Comes as either Community Edition (CE) and Enterprise Edition (EE).

Check Version

Version should be listed at /help, but version # may not appear unless you're signed in. May be able to create an Acct to view.

Revision

Source Code may contain a line like gon.revision="7f1a7c62df9". Search for “revision” since it's buried in there. The Revision may indirectly lead you to a Ver #.

Searching gitlab "7f1a7c62df9" got forum posts / issues which contained lines like GitLab information Version: 15.6.0 Revision: 7f1a7c62df9

User Enum

Depending on the version, users may have their own page at /, so can check if a pg exists to determine if there's a user with that name (e.g /bob). User ID of 1 means it's admin.

Diff methods for diff versions with automated scripts for some.
